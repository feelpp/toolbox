== Description

In this example, we simulate the oscillating motion of a highly flexible 2D pendulum. 

== Running the case

The command line to run this case is


[[command-line2D]]
[source,sh]
----
mpirun -np 4 feelpp_toolbox_solid --case "github:{repo:toolbox,path:examples/modules/csm/examples/pendulum/pendulum2D} 
----

== Geometry

The pendulum is an isosceles triangle-shaped mesh.
We use a discretization of order 2.

image:pendulum/mesh2D.png[50%]

The mesh can be retrieved from link:https://girder.math.unistra.fr/api/v1/file/5b200dcbb0e9570499f66bfb/download[Girder] 
(see the girder xref:contribute:girder:README.adoc[documentation]).

==  Boundary conditions

=== Vertex angle

The only boundary condition is a Dirichlet one: the pendulum is held by its vertex angle while it oscillates.
Hence, the displacement of this vertex is zero at all times.
