== Description

In this example, we simulate a fluid evolving in a Cyclone mesh using the Stokes equation.


== Running the case

The command line to run this case is

[[command-line]]
[source,sh]
----
mpirun -np 4 feelpp_toolbox_fluid --case "github:{repo:toolbox,branch:case-cyclone-cfd,path:examples/modules/cfd/examples/cyclone}"
----

=== Geometry

image:cyclone/mesh.png[50%]
	
The mesh can be found on link:https://girder.math.unistra.fr/api/v1/item/5b1eaaffb0e9570499f66ba8/download[Girder]
 

== Input parameters

[options="header"]
|===
| Name |Description | Value | Unit |
| stem:[mu_f] | fluid velocity | 1e-3  | stem:[Pa.s] |
| stem:[rho] | fluid density | 998 | stem:[kg/m^3] |
|===


== Model & Toolbox

The model used is a Stokes model. It is described in the CFD toolbox, see xref:toolboxes:cfd:index.adoc[Computational Fluid Dynamics]
