= Cyclone
:page-vtkjs: true

== Description of the example

In this example, we simulate a fluid evolving in a Cyclone mesh using the Stokes equation.

== Geometry

image:cyclone/mesh.png[50%]


== Input parameters

[options="header"]
|===
| Name |Description | Value | Unit |
| stem:[mu_f] | fluid velocity | 1e-3  | stem:[Pa.s] |
| stem:[rho] | fluid density | 998 | stem:[kg/m^3] |
|===


== Model & Toolbox

The model used is a Stokes model. It is described in the CFD toolbox, see xref:toolboxes:cfd:index.adoc[Computational Fluid Dynamics]

To reproduce those results, one should run either the first line for the linear case, or the second one for the non linear case:
[source,shell]
----
mpirun -np 4 feelpp_toolbox_fluid_3d --case "github:{path:toolboxes/CFD/cyclone/cyclone}"
----
