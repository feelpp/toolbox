== Description

In this example, we simulate a fluid evolving in a spring mesh using the Stokes equation.

== Running the case

The command line to run this case is
[[command-line]]
[source,sh]
----
mpirun -np 4 feelpp_toolbox_fluid --case "github:{repo:toolbox,path:examples/modules/cfd/examples/spring}"
----

== Geometry

The geometry consists of a spring shaped tube with one inlet and one outlet.

image:spring/mesh.png[50%]


== Input parameters

[options="header"]
|===
| Name |Description | Value | Unit |
| stem:[\mu_f] | fluid viscosity | 6e-3  | stem:[Pa.s] |
| stem:[\rho] | fluid density | 1056 | stem:[kg/m^3] |
|===

== Results

The fields of interest for this example are the velocity, the pressure of the fluid and the parallel process id (pid). The pid helps to see how the mesh was partitioned for parallel processing. In the 3D view below, when selecting the pid in the first dropdown list, each color corresponds to a subdomain assigned to and processed by a CPU core.
